# Motion-Tracking

This assignment investigates human motion tracking during walking using tri-dimensional acceleration data obtained via a smartphone.
The main goal is to analyze the movement patterns of the user and use them to predict the location of the phone on the body. Specifically, the task focuses on classifying the phone position among four possible locations:

  - Left hand
  - Right hand
  - Left trouser pocket
  - Right trouser pocket
